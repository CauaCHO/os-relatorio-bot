# 🤖 Bot de Relatório O.S.

Bot desenvolvido em Python para automatizar o preenchimento de relatórios técnicos de Ordens de Serviço diretamente pelo Telegram.

---

## 🚀 Funcionalidades

- 📋 Coleta de dados via chat (fluxo guiado)
- 🧠 Correção automática de texto com IA (OpenAI)
- 📝 Geração de relatório técnico padronizado
- 📤 Envio automático do relatório para grupo
- ⚙️ Adaptação dinâmica conforme tipo de O.S.
- ⏱️ Cálculo automático do tempo de atendimento
- 🧾 Formatação profissional com negrito no Telegram

---

## 🛠️ Tecnologias utilizadas

- Python
- python-telegram-bot
- OpenAI API
- Git / GitHub

---

## 📱 Como funciona

1. O técnico inicia o bot com `/start`
2. O bot faz perguntas guiadas
3. O técnico responde no chat
4. O bot gera o relatório automaticamente
5. O relatório é enviado para o grupo da equipe
